# Security Policy

## Supported Versions
| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability
Please email **security@pcnaid.com** or open a private security advisory
(<Repo ▸ Security ▸ Advisories ▸ New Draft Advisory>).  
We aim to respond within **72 hours**.
